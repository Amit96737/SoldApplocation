# 🛒 E-Commerce Mobile Application (FastAPI)

This is a feature-rich **E-commerce Mobile Application Backend** built using **FastAPI**. The platform allows users to **buy, sell, and rent items**, providing a flexible marketplace experience.

---

## 🚀 Features

* 🛍️ Users can **add items** to the platform
* 💰 Users can **sell items**
* 🛒 Users can **buy items**
* 📦 Users can **rent out their items**
* 🔐 Secure authentication system
* 💳 Integrated **AllPay payment gateway** for transactions

---

## 🛠️ Tech Stack

* **FastAPI** – Backend framework
* **Alembic** – Database migrations
* **SQLAlchemy** – ORM
* **Swagger UI** – API documentation
* **AllPay** – Payment integration

---

## 🧑‍💼 Admin Panel

The application includes a powerful **Admin Panel** with the following capabilities:

* View all platform data
* Track:

  * Items added
  * Items sold
* Manage:

  * Categories
  * Subcategories
  * Sizes
  * Brands
  * Materials

---

## 📄 API Documentation

* Interactive API documentation is available via **Swagger UI**
* Makes it easy to test and explore endpoints

---

## 🔧 Key Functionalities

* User-based item management (add, update, delete)
* Marketplace system (buy/sell/rent)
* Payment processing via AllPay
* Admin-controlled taxonomy (categories, brands, etc.)
* Clean and scalable backend architecture

---

## 📌 Summary

This project is a complete backend solution for a modern **E-commerce mobile application**, supporting multiple business models like **selling, buying, and renting**, with secure payments and full admin control.
